# 拼圖遊戲 (Puzzle Game)

這個簡單的拼圖遊戲專為小朋友設計：只要上傳一張照片，就能立即把圖片切成 4x4 的拼圖，透過拖放或觸控在同一個格子中移動拼圖塊，還原原始畫面。

## 特色

- **單一響應式格子**：無論手機或電腦，拼圖區都會自動縮放，保持可操作與可視。
- **直覺拖放與觸控支援**：用滑鼠或手指都能輕鬆拖動拼圖到正確位置。
- **完成提示**：每完成一格，會顯示進度；完成全部後會彈出恭喜訊息。

## 使用方式

1. 點選「📷 上傳照片」選擇圖片（可用自己畫的圖或喜歡的照片）。
2. 拖動拼圖塊到空格中，直到每塊恢復原本位置。
3. 若想再玩一次，按左上角的「🔀 重新打亂」。
4. 完成會看到「🎉 恭喜完成！ 🎉」訊息，可以再玩一次。

## 本地測試

1. 打開 `index.html` 就能在瀏覽器直接遊玩。
2. 建議使用最新版本的 Chrome/Safari/Edge 以確保拖放與觸控最順暢。
